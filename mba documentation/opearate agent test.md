## POST /

Run an agent test

Send a test message to the AI agent and receive a response. Messages are processed through the full agent pipeline without requiring a consumer phone number.


### Header Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| X-API-Version | "2.0.0" |  |  |

### Path Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| entity_id | string | ✓ | The WhatsApp Business Phone Number ID for the AI agent to test. |

### Request Body (Required)

**Content Type**: `application/json`

**Schema**: [BizAIOmniChannelAgentTestRequest](#bizaiomnichannelagenttestrequest)

### Responses

**200**

Response containing the AI agent response and test conversation metadata


**Content Type**: `application/json`

**Schema**: [BizAIOmniChannelAgentTestResponse](#bizaiomnichannelagenttestresponse)

**400**

Bad request

**Content Type**: `application/json`

**Schema**: [StandardError](#standarderror)

**401**

Unauthorized

**Content Type**: `application/json`

**Schema**: [StandardError](#standarderror)

**429**

Too many requests

**Content Type**: `application/json`

**Schema**: [StandardError](#standarderror)

**500**

Server error

**Content Type**: `application/json`

**Schema**: [StandardError](#standarderror)


# Components

## Schemas

<jumplink id="bizaiomnichannelagenttestrequest"></jumplink>
### BizAIOmniChannelAgentTestRequest

| Property | Type | Required | Description |
|----------|------|----------|-------------|
| user_msg | string | ✓ | The text content of the test message to send to the AI agent |
| conversation_id | string |  | Identifier for multi-turn test conversations. Provide the conversation_id from a previous response to continue that conversation. |

<jumplink id="bizaiomnichannelagenttestresponse"></jumplink>
### BizAIOmniChannelAgentTestResponse

| Property | Type | Required | Description |
|----------|------|----------|-------------|
| message_id | string | ✓ | Unique identifier for this message exchange |
| agent_response | string | ✓ | The AI agent response text |
| conversation_id | string | ✓ | The test conversation identifier. Use this in subsequent requests for multi-turn conversations. |
| timestamp | integer |  | Unix timestamp when the response was generated |
| handoff_reason | string |  | If the agent hands off to a human, this contains the reason |
| no_response_reason | string |  | If the agent did not generate a response, this contains the reason. Possible values include ELIGIBILITY_CHECK_FAILED. |
| quick_replies | array of string |  | Suggested quick reply messages generated by the AI agent |
| product_variant_ids | array of string |  | The variant IDs of the products referenced in the agent response |
