## DELETE /{file_id}

Delete a knowledge file

Delete a specific file from the knowledge base by its ID


### Header Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| X-API-Version | "2.0.0" |  |  |

### Path Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| entity_id | string | ✓ | The WhatsApp Business Phone Number ID for the Meta Business Agent. |
| file_id | string | ✓ | The unique identifier of the file |

### Responses

**204**

**404**

Not found

# Components

## Schemas

<jumplink id="bizaiomnichannelknowledgefilerequest"></jumplink>
### BizAIOmniChannelKnowledgeFileRequest

| Property | Type | Required | Description |
|----------|------|----------|-------------|
| file_name | string | ✓ | The name of the file being uploaded |
| file | string (binary) | ✓ | The file to be uploaded. Maximum file size: 100000000 bytes. Supported file types: - .pdf - .doc - .docx - .png - .jpg - .jpeg - .csv, when CSV extraction is enabled for the WhatsApp asset - .xlsx, when XLSX extraction is enabled for the WhatsApp asset |

<jumplink id="bizaiomnichannelknowledgefileresponse"></jumplink>
### BizAIOmniChannelKnowledgeFileResponse

| Property | Type | Required | Description |
|----------|------|----------|-------------|
| id | string | ✓ | The unique identifier for this file |
| file_name | string | ✓ | The name of the file |
